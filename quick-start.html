Quick Start.

Getting Started

01. INSTALLATION

Install DyGram via npm:

npm install

02. DEVELOPMENT

Start the development server:

npm run dev

03. CLI USAGE

Execute .mach files directly:

npx dygram your-file.mach

Your First Machine

Create a file called hello.mach with the following content: 

machine "Hello World"

state start;
state process;
state end;

start -> process -> end;

Run it: 

npx dygram hello.mach
