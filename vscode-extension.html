VS Code Extension.

Editor Integration

Full DyGram language support in Visual Studio Code.

SYNTAX HIGHLIGHTING

Beautiful, semantic syntax highlighting for .mach files with support for:

  • Keywords and entity types
  • Relationships and connections
  • Type annotations
  • String interpolation
  • Comments

INTELLISENSE

Smart code completion powered by Langium LSP:

  • Entity type suggestions
  • Attribute completions
  • Relationship labels
  • Type annotations
  • Context-aware suggestions

REAL-TIME VALIDATION

Instant feedback on syntax and semantic errors:

  • Syntax error detection
  • Type checking
  • Undefined reference warnings
  • Quick fixes
  • Inline diagnostics

DIAGRAM PREVIEW

Visualize your DyGram models as Mermaid diagrams directly in the editor:

  • Live diagram generation
  • Side-by-side view
  • Auto-refresh on save
  • Export to PNG/SVG

CODE NAVIGATION

Navigate large DyGram files with ease:

  • Go to definition
  • Find all references
  • Symbol outline
  • Breadcrumb navigation

FORMATTING

Keep your code clean and consistent:

  • Auto-formatting on save
  • Configurable indentation
  • Code folding
  • Comment toggling

Installation

The extension is built into the project. To develop with it: 

# Clone the repository
git clone https://github.com/christopherdebeer/machine.git

# Install dependencies
npm install

# Build the extension
npm run build

# Open in VS Code
code .

Usage

Create a new file with the .mach extension and start typing. The extension will automatically activate and provide language support.

Press F5 to launch a development instance of VS Code with the extension loaded.

Technology

Built with modern language server technology:

LANGIUM

Powerful language engineering framework that provides the foundation for all language features.

LSP

Language Server Protocol ensures consistent behavior across all editors that support LSP.

MONACO

VS Code's editor component provides rich editing capabilities in the browser playground.
