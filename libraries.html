Libraries.

Built With

DyGram is powered by best-in-class open source libraries.

LANGIUM

Language engineering framework that powers the DSL, parser, and Language Server Protocol implementation.

langium.org →

MONACO EDITOR

The code editor that powers VS Code, providing rich IntelliSense and editing features in the desktop playground.

monaco-editor →

CODEMIRROR 6

Lightweight, extensible code editor optimized for mobile devices, powering the mobile playground experience.

codemirror.net →

VITE

Next-generation frontend build tool that provides lightning-fast development server and optimized production builds.

vitejs.dev →

MERMAID

Markdown-inspired diagram generation that creates beautiful visualizations from DyGram state machines.

mermaid.js.org →

ESBUILD

Extremely fast JavaScript bundler used for CLI and library distribution builds.

esbuild.github.io →

Development Dependencies

Additional tools that make DyGram development productive:

TYPESCRIPT

Type-safe development with full IntelliSense and compile-time error checking.

VITEST

Blazing fast unit testing framework with native TypeScript support.

ESLINT

Code quality and style enforcement to maintain consistent, clean code.

CHEVROTAIN

Parser building toolkit used by Langium for robust syntax analysis.

VSCODE API

Extension API for building first-class VS Code integration.

NODE.JS

Runtime environment for CLI tools and language server execution.

Why These Libraries?

Each library was chosen for its excellence in a specific domain:

  • Langium - Purpose-built for language engineering with LSP built-in
  • Monaco - Industry-leading editor with proven reliability
  • CodeMirror - Best mobile editing experience available
  • Vite - Fastest development experience with modern ES modules
  • Mermaid - Beautiful diagrams without heavy dependencies
  • Esbuild - Unmatched bundling speed for distribution
