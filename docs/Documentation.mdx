## Complete Documentation

Welcome to the DyGram documentation. DyGram is a dynamic prototyping platform featuring **Machine**, a lean DSL for rapid system design.

---

## Understanding the Name

- **DyGram** = The complete project (playground, tooling, VS Code extension)
- **Machine** = The DSL (Domain-Specific Language) itself
- **File Extensions**: Use `.dygram` for Machine language files (legacy `.mach` also supported)
